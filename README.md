# SamQuiz AI - AI 퀴즈 자동 생성 도구

PDF나 이미지를 업로드하면 **AI가 자동으로 퀴즈를 만들어주는 도구**입니다.

---

## 이 도구는 무엇인가요?

샘퀴즈 어드민에서 퀴즈를 만들 때, 일일이 문제를 입력하는 대신 **AI가 자동으로 문제를 생성**해줍니다.

### 현재 상태

> **PoC(개념 검증) 단계**입니다.
>
> 향후 샘퀴즈 어드민에 정식으로 개발되기 전, **기능에 대한 피드백을 수집**하기 위한 목적으로 제작되었습니다.
> 사용 중 불편한 점이나 개선 아이디어가 있다면 알려주세요!

---

## 사용 방법

### 1단계: 북마크 등록하기

1. 브라우저에서 **북마크 바**를 표시합니다
   - Chrome: `Ctrl + Shift + B` (Windows) / `Cmd + Shift + B` (Mac)
2. 북마크 바에서 **마우스 우클릭** → **페이지 추가** 선택
3. 이름: `SamQuiz AI` (원하는 이름으로)
4. URL에 아래 코드를 **전체 복사**해서 붙여넣기:

```
javascript:(function(){var s=document.createElement('script');s.src='https://cdn.jsdelivr.net/gh/viyoosang/samquiz_ai@main/v18-modules/dist/samquiz-ai-chatbot-v18.js';document.body.appendChild(s);})();
```

5. **저장** 클릭

### 2단계: 퀴즈 생성하기

1. [샘퀴즈 어드민](https://samquiz.vivasam.com/makeQuiz) 페이지를 엽니다
2. 방금 만든 **SamQuiz AI 북마크**를 클릭합니다
3. 화면 오른쪽 아래에 **챗봇 아이콘**이 나타납니다
4. 아이콘을 클릭하면 AI 퀴즈 생성 패널이 열립니다

### 3단계: 문제 생성 및 적용

1. **PDF나 이미지**를 업로드합니다 (교과서, 지도서 등)
2. 원하는 **문제 유형**을 선택합니다
3. **문제 생성하기** 버튼을 클릭합니다
4. AI가 문제를 생성하면, **폼에 적용** 버튼으로 자동 입력합니다

## 지원하는 문제 유형

| 유형 | 설명 |
|------|------|
| 선택형 | 4개 보기 중 정답 선택 |
| OX형 | O 또는 X 선택 |
| 단답형 | 짧은 답 입력 |
| 서술형 | 긴 답변 작성 |
| 순서형 | 항목을 순서대로 배열 |
| 초성퀴즈 | 초성 힌트로 정답 맞추기 |
| 글자순서바꾸기 | 섞인 글자로 단어 맞추기 |
| 가로세로퍼즐 | 낱말 퍼즐 |

## 자주 묻는 질문

### Q: 북마크를 클릭해도 아무 반응이 없어요
브라우저 주소창에 `javascript:`로 시작하는 코드가 제대로 저장되었는지 확인해주세요.
일부 브라우저는 붙여넣을 때 `javascript:` 부분을 자동으로 삭제하므로, 다시 한번 확인이 필요합니다.

### Q: API 키는 어디서 받나요?
Google AI Studio에서 Gemini API 키를 발급받을 수 있습니다.
설정 패널에서 API 키를 입력하면 됩니다.

### Q: 생성 중간에 멈추고 싶어요
문제 생성 중에 **중단** 버튼을 클릭하면 작업이 취소됩니다.

### Q: 생성된 문제가 마음에 안 들어요
같은 자료로 다시 생성하면 다른 문제가 만들어집니다.
또는 "추가 지시" 입력란에 원하는 스타일을 적어주세요.

```
├── src/

├── build.js         # 빌드 스크립트
└── dist/            # 빌드 결과물
```
