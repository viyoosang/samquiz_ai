# SamQuiz AI

샘퀴즈 어드민 저작툴에서 작동하는 **AI 기반 퀴즈 자동 생성 도구**입니다.

## 사용 방법

### 북마크릿으로 사용하기

아래 코드를 북마크 URL에 등록하세요:

```javascript
javascript:(function(){var s=document.createElement('script');s.src='https://cdn.jsdelivr.net/gh/viyoosang/samquiz_ai@main/v18-modules/dist/samquiz-ai-chatbot-v18.js';document.body.appendChild(s);})();
```

### 직접 실행하기

1. [샘퀴즈 어드민](https://samquiz.vivasam.com/makeQuiz) 페이지 열기
2. F12 개발자 도구 > Console
3. `v18-modules/dist/samquiz-ai-chatbot-v18.js` 내용 붙여넣기
4. 화면 우측 하단 챗봇 아이콘 클릭

## 주요 기능

- **PDF/이미지 업로드**: 교과서/지도서 PDF를 업로드하여 내용 기반 문제 생성
- **8가지 문제 유형 지원**: 선택형, OX형, 단답형, 서술형, 순서형, 초성퀴즈, 글자순서바꾸기, 가로세로퍼즐
- **학교급별 설정**: 초등/중고등 난이도 자동 조절
- **유사정답 자동 생성**: 단답형에서 동의어/표기 변형 자동 추가
- **폼 자동 입력**: 생성된 문제를 샘퀴즈 폼에 자동 입력

## 지원 문제 유형

| 유형 | 설명 |
|------|------|
| 선택형 | 4지선다 객관식 |
| OX형 | O/X 이지선다 |
| 단답형 | 짧은 답 입력 (유사정답 지원) |
| 서술형 | 모범답안 + 채점기준 |
| 순서형 | 항목 순서 배열 |
| 초성퀴즈 | 초성 힌트로 정답 맞추기 |
| 글자순서바꾸기 | 글자 순서를 바꿔 단어 맞추기 |
| 가로세로퍼즐 | 가로세로 낱말 퍼즐 |

## 기술 스택

- **LLM API**: Google Gemini 3.0 Flash Preview
- **프론트엔드**: Vanilla JavaScript (IIFE 패턴)
- **타겟 환경**: 샘퀴즈 어드민 저작툴 (Vue.js 기반)

## 개발

### 빌드

```bash
cd v18-modules
node build.js
```

빌드 결과물: `v18-modules/dist/samquiz-ai-chatbot-v18.js`

### 프로젝트 구조

```
v18-modules/
├── src/
│   ├── config.js    # 설정 및 프롬프트
│   ├── utils.js     # 유틸리티 함수
│   ├── state.js     # 상태 관리
│   ├── styles.js    # CSS 스타일
│   ├── templates.js # HTML 템플릿
│   ├── api.js       # Gemini API 호출
│   ├── form.js      # 폼 자동 입력
│   └── ui.js        # UI 이벤트 핸들링
├── build.js         # 빌드 스크립트
└── dist/            # 빌드 결과물
```
